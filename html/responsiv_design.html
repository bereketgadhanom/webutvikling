Responsiv-design

Responsiv design er en nettside design som brukes på forskjellige enhetsskjerm størrelser. Responsiv nettside endres automatisk til å tilpasse for den tilgjengelig enhetsskjerm. Elementene i nettsiden kan krympes, flyttes, skjules eller utvides for å tilpasse skjermstørrelser. Siden bruk av smart telefoner og nettbretter blir større og større er det veldig viktig å bruke responsiv design. Slik at mobil og nettbrett brukere kan se nettsiden enkelt.
Responsiv nettside bygges opp av funksjoner media queries, fleksibelt størrelser (grid og flex) og mediatyper.

Media queries

Media querry er en av de viktigste funksjoner for en responsiv nettside design. Media query er en CSS-modul som tillater innhold av nettsiden til å tilpasse seg i forhold til enhetsskjermoppløsning. Fore eksempel smarttelefonskjerm, nettbrett og pc-skjerm.

Media typer

Medietyper kan du spesifisere en type media som skal målrettes, slik at du kan målrette utskrift, håndholdt og så videre. Media type er en funksjon for å spesifisere hvordan dokumenter presentere i forskjellige medier. Media refererer til enhetstyper som viser nettsiden for eksempel pc-skjerm, nettbrett og smarte telefoner. Her er liste av noen css media typer. All: passer på alle media enheter og er standard innstilt. Print: brukes for skrivere. Screan: brukes for pc-skjerm, nettbrett og smarte mobile enheter. Speech: brukes for skjermleser. Man kan bruke de logiske operatorene AND, ONLY and NOT.

Fleksibel størrelser

Flexbox (Flexible Box Layout Module) er laget for å gjøre det enkelt for oss å legge ting ut som en rad eller som en kolonne. For å bruke flexbox, bruker du "display: flex" til overordnet elementet av elementene du vil legge ut. Alle sine direkte barn blir deretter fleksible elementer. Grid: mens flexbox er designet for et dimensjonalt layout, er Grid Layout designet for to dimensjoner. Det betyr at det kan ordres på rader og kolonner. Grid Layout gir oss en metode for å lage grid strukturer som er beskrevet i CSS. Det hjelper oss med å lage layout som kan omdefineres. Grid Layout lar oss skille ordrenes elementer i kilden riktig fra sin visuelle presentasjon
