CSS Layout

CSS-layout er et utseende av en nettside som inneholder dokument, bilde, tekst eller annet som er utformet for å være mer attraktivt for brukere og hjelpe dem med å identifisere hva de ser. CSS-layout teknikker tillater oss å ta elementer inneholdt i en nettside og kontrollere hvor de er plassert i forhold til deres standardposisjon. Det er flere CSS-layout typer. Alle har egen fordeler og ulemper. Derfor må man ha god kunnskap å bruke dem effektivt.

CSS-layout metoder:

  • Flexbox (Flexible Box Layout Module) er laget for å gjøre det enkelt for oss å legge ting ut som en rad eller som en kolonne. For å bruke flexbox, bruker du "display: flex" til overordnet elementet av elementene du vil legge ut. Alle sine direkte barn blir deretter fleksible elementer.
  • Grid: mens flexbox er designet for et dimensjonalt layout, er Grid Layout designet for to dimensjoner. Det betyr at det kan ordres på rader og kolonner. Grid Layout gir oss en metode for å lage grid strukturer som er beskrevet i CSS. Det hjelper oss med å lage layout som kan omdefineres. Grid Layout lar oss skille ordrenes elementer i kilden riktig fra sin visuelle presentasjon
  • Floats: er en CSS-posisjoneringsegenskap og har verdier left, right, inherit og none. Med float et element endrer oppførselen til elementet og blokk nivå elementene som følger det i normal flow. Elementet flyttes til venstre eller høyre og fjernes fra normal flow. For å forstå formålet kan vi se på å skrive ut design. I et print layout kan bilder settes inn på siden slik at teksten brytes rundt dem etter behov.
  • Table-layout: har verdier auto og fixed. HTML-tabeller er bra for å vise tabelldata. men å bruke den som css er ikke anbefales. Tabelloppsett er ufleksible, vanskelig å feil søke og semantisk feil.
  • Framework: er en pakke som består av en struktur av filer og mapper med standardisert kode (HTML, CSS, JS dokumenter etc.) som kan brukes til å støtte utviklingen av nettsteder.
