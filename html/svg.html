SVG

SVG står for Scalable Vector Graphics. SVG er xml (Extention Marcup Language) basert filformat for markeringsspråk og brukes til å tegne og beskrive todimensjonal vektorgrafikk. SVG filformatet er skalerbart og implementert i mer enn en nettleser. Grafikk som er lagret i SVG-format, er ikke avhengig av lys og oppløsning. Denne grafikken kan dermed skaleres til alle størrelser uten å påvirke områdets utseende eller tiden det tar å laste det inn. Vektorgrafikk lagrer matematisk formler istedenfor pixler.

Med SVG kan man lage logoer, ikoner, animasjoner, illustrasjoner og grafer. Man kan lage SVG med bildebehandlingsprogrammer som støtter vektorgrafikk som adobe illustrator, Sketch og Inkscape eller man kan lage med egen kode. Man kan bruke svg ved å bruke object elementet. Man kan lage SVG direkte i html ved å bruke xml elementer som rect, circle ellipse, line, polygon, polyline, path, text OSV Eksempel på Vektorgrafikk er swf, svg, pdf osv.

SVG tillater tre typer grafiske objekter: vektor grafiske former (for eksempel stier som består av rette linjer og kurver), bilder og tekst. Grafiske objekter kan grupperes, formateres, forvandles og sammensettes i tidligere gjengitte objekter. Funksjonssettet inkluderer nestede transformasjoner, utklippsstier, filtereffekter og mal objekter. SVG-basert grafikk kan inkluderes rett inn i HTML-dokumentene. Grafikken kan gjøres dynamisk ved hjelp av JavaScript, på samme måte som med HTML, og vi kan også gjøre endringer i utseendet ved hjelp av CSS.

Fordeler

  • Vektorgrafikk skaleres uten å endre oppløsningen eller kvaliteten.
  • Tar mindre plass enn punktgrafikk
  • Redigeres enkelt.
  • Vektor grafikk kan enkelt konvertere til punktgrafikk (eks. JPEG, GIF, PNG).

Ulemper

  • SVG støtter ikke på alle versjoner av nettlesere.
  • Vanskelig å konvertere andre formater til SVG.
  • Kan ikke brukes til bilder.
