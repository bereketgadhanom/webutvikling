CSS Media Type

CSS har viktige funksjoner for å spesifisere hvordan dokumenter presentere i forskjellige medier. Media refererer til enhetstyper som viser nettsiden for eksempel pc-skjerm, nettbrett og smarte telefoner. 

      Eksempel på hvordan
        man bruker medeiene 
        @media print {
           body {font-size: 10px;}
          }
          @media screen{
           body {font-size: 12px;}
          }
          @media screen, print{
           body {line-height: 1.0;}
          }
        
        Her er liste av noen css
         mediatyper: 
        -All: passer på alle media enheter
         og er standard innstilt.
        -Print: brukes for skrivere.
        -Screan: brukes for pc-skjerm,
          nettbrett og smarte mobile enheter.
        -brukes for skjermlesere som
          tale-leser skjermen.
